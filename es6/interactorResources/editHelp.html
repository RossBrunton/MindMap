Working with objects

  • Click and drag to move objects
  • Click an object to edit the object
  • Double click on an empty space to create a new object

Working with arrows

  • Click an arrow to edit the arrow
  • Drag from the green dot on a node to create a new arrow
  • Drag an arrow to create a point for it to curve around
  • Drag the green point on an arrow to change the curve point
  • Drag the green ends of an arrow to change the connections of the arrow

Other

  • Go to the edge and click an arrow to resize your drawing area

Multiple selection

  • Hold shift and click to select multiple objects
  • Drag with the right mouse button to also select multiple objects
  • Press the escape key or click anywhere to clear multiple selection
  • Press the delete key to delete multiple nodes that are selected

Keyboard Shortcuts

  • CTRL+A: Select All
  • CTRL+Z: Undo
  • CTRL+R: Redo
